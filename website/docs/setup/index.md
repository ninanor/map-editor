# Setup

This section covers everything you need to get NINA Maps running, from local development to production deployment with Docker.

## Prerequisites

| Requirement | Minimum version |
|---|---|
| Node.js | 18+ |
| pnpm | latest |
| Modern browser | WebGL support required |

Install pnpm globally if you don't already have it:

```bash
npm install -g pnpm
```
